# KnowledgeBase
Features
Administrator
- Manage library information (name, address, contact, opening hours, loan duration).  
- Add, modify, delete, search, and display **students, teachers, and books**.  
- Each book has a unique ID, title, author, category, year, and available copies.  
- Access a **dashboard** showing:
  - Available books and copies  
  - Borrowed books  
  - Users with borrowed items  
 Student
- Register with personal details (name, ID, field, level, contact info).  
- Search and view available books.  
- Borrow books (if copies are available).  
- Return books (updates availability).  
- View personal account and borrowed books list.  
 Teacher
- Register with personal details (name, department, grade, contact info).  
- Search and view available books.  
- Borrow books (within a defined limit).  
- View account and borrowing history.  

Technical Highlights
- Implemented in **C language** using:
  - Loops, conditionals, arrays, functions, pointers, and dynamic memory allocation.  
  - **Structs** for user and book records.  
  - File handling for persistent storage.  
- Console menus with clear data presentation.
